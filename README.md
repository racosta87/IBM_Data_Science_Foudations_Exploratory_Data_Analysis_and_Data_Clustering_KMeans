# IBM Data Science Foundations - Statistics and Clustering in Python Course
## Exploratory Data Analysis and Data Clustering with KMeans

This project covers exploratory data analysis (EDA) and data clustering using the KMeans algorithm, as part of the IBM Data Science Foundations course. The dataset used in 
this exercise is derived from the Banknote Authentication problem, a well-known dataset in the field of fraud detection and machine learning from 
[OpenML](https://www.openml.org/search?type=data&sort=runs&id=1462)

## Tools used
- Python
- Jupyter Notebook
- Pandas, Matplotlib, Scikit-learn
